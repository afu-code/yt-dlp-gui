## [Unreleased]

### Added
- Initial release of yt-dlp-gui
- Modern graphical user interface for yt-dlp
- Multi-language support (English, Chinese Simplified, Chinese Traditional, Japanese, Korean)
- Tabbed interface for organized settings:
  - General: Format selection, quality, output templates
  - Network: Proxy, cookies, rate limiting
  - Filters: Playlist filters, date ranges
  - Post-Processing: Audio extraction, video conversion, metadata embedding
  - Advanced: Extra CLI arguments, command preview
- Settings window with:
  - Output directory configuration
  - FFmpeg path detection and configuration
  - Theme selection
  - Language selection
- Real-time command preview
- Download progress tracking
- Comprehensive logging
- Cross-platform support (Windows, Linux, macOS)
